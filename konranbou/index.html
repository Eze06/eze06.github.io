KONRANBOU

Overview

Konranbou is a 7-week international game project built by a cross-cultural team. The game is a co-op roguelike experience inspired by titles such as It Takes Two and Split Fiction, focusing on cooperative gameplay and shared progression. As the lead programmer, I was responsible for overseeing the game’s technical development, implementing core systems, and working closely with the team to deliver the project within a short timeline. This project strengthened both my programming skills and my ability to collaborate effectively in an international development environment.
